# 🔊 سافت‌ساوند — تجربه‌ی صدای محیطی

---

## ⚡ جزئیات فنی

- **معماری صدا**: موتور پخش چندلایه با پشتیبانی از چند track همزمان و کنترل مستقل حجم و fade برای هر کانال
- **مدیریت state**: Redux Toolkit برای مدیریت وضعیت کلی میکس — track‌ها، حجم‌ها و همگام‌سازی پخش بین کامپوننت‌ها
- **انیمیشن‌ها**: Framer Motion برای انتقال‌های روان، تعامل با کارت‌های صدا و بازخورد بصری محیطی
- **دسترس‌پذیری**: رابط کاربری سازگار با WCAG — قابل ناوبری با کیبورد، سازگار با screen reader، با احترام به تنظیمات reduced-motion
- **SEO**: یکپارچه‌سازی next-sitemap با متادیتای ساختارمند
- **پرفورمنس**: Lighthouse بالای ۹۰ از طریق lazy loading دارایی‌های صوتی، code splitting و بهینه‌سازی bundle
- **واکنش‌گرایی**: طراحی mobile-first، کاملاً کاربردی در تمام اندازه‌های صفحه

## 📁 ساختار پروژه

```
src/
├── app/                 # مسیرهای App Router
│   ├── page.tsx         # صفحه اصلی / پلیر
│   └── blog/            # بخش وبلاگ
├── components/          # کامپوننت‌های قابل استفاده مجدد
│   ├── player/          # کامپوننت‌های پلیر صدا
│   └── ui/              # کامپوننت‌های پایه shadcn/ui
├── store/               # Redux Toolkit slices (میکس، پخش)
├── data/                # تعریف track‌های صوتی (JSON)
└── public/              # دارایی‌های استاتیک صوتی و تصویری
```

## ⚙️ نحوه اجرا در محیط محلی

```bash
# کلون کردن پروژه
git clone https://github.com/mostafakm78/murmur-sounds-web.git

# ورود به پوشه پروژه
cd murmur-sounds-web

# نصب پکیج‌ها
npm install

# اجرای سرور توسعه
npm run dev
```

سپس پروژه از آدرس [http://localhost:3000](http://localhost:3000) در مرورگر در دسترس خواهد بود.
