## 📦 اجرای پروژه در محیط محلی

برای اجرای پروژه به‌صورت محلی مراحل زیر را دنبال کنید:

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
npm install
npm run dev
```

سپس مرورگر را باز کرده و وارد آدرس زیر شوید:  
[http://localhost:3000](http://localhost:3000)

صفحه‌ی اصلی در مسیر زیر قرار دارد:  
`src/app/page.tsx`

---

## 📁 ساختار پوشه‌های پروژه

```
.
├── src/app/            # مسیر اصلی اپلیکیشن Next.js
│   └── page.tsx        # صفحه اصلی
├── components/         # کامپوننت‌های قابل استفاده مجدد
├── public/             # فایل‌های استاتیک (تصاویر و ...)
├── data/               # فایل‌های داده JSON
├── tailwind.config.ts  # تنظیمات Tailwind
├── tsconfig.json       # پیکربندی TypeScript
```

## 🛠️ نحوه‌ی استقرار پروژه

این پروژه روی **Vercel** مستقر شده است.  
برای راه‌اندازی نسخه‌ی شخصی خودتان:

1. پروژه را در گیت‌هاب پوش کنید.  
2. آن را به حساب کاربری خود در Vercel متصل کنید.  
3. مسیر ریشه (Root Directory) را انتخاب کرده و Deploy بزنید.
